# numpy functions

```python
import numpy as np
```

np.zeros
-
    모든 배열의 원소가 0인 배열을 생성
    
np.ones
-
    모든 배열의 원소가 1인 배열을 생성
    
np.empty
-
    배열의 크기만 정한 배열을 생성
    원소는 쓰레기 값(매우 크거나 작은 값)이 들어가 있다.

np.arange
-
    수열의 원소를 가진 배열을 생성
    range()와 유사한 형태
    
np.full
-
    배열의 모든 값이 하나의 상수인 배열을 생성

np.eye
-
    지정한 크기의 단위행렬을 생성

np.random.random
-
    배열의 원소를 임의의 값으로 생성
    값: 0. ~ 1.
